📚ThreatFlow-ML
Clasificación Multiclase de Incidentes de Ciberseguridad con Machine Learning ThreatFlow ML es un sistema de clasificación multiclase de incidentes de ciberseguridad basado en Machine Learning. El modelo utiliza características de tráfico de red del conjunto de datos UNSW-NB15 para predecir la categoría de ataque asociado a cada flujo de red.
